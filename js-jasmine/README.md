# Gilded Rose Javascript Tech Test

## User Stories
```
As an employee
I'd like there to be a Shopfront
So that we can keep Items on display

As an employee
I'd like Items to have a SellIn value
So that we know how long to keep it on display

As an employee
I'd like each Item to have a Quality value
So that I know how valuble each Item is

As an employee
I'd like to recalculate an Item's Quality value at the end of the day
So that I know what items to display tomorrow
```
